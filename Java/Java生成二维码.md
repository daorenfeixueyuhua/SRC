## 4. 二维码优缺点
### 4.1 二维码的优缺点
> **优点：**
> 高密度编码，信息容量大
> 编码范围广
> 容错能力强
> 译码可靠性高
> 可引入加密措施
> 成本低、易制作、持久耐用

> **缺点:**
> 二维码技术成为手机病毒、钓鱼网站传播的新途径
> 信息泄露
